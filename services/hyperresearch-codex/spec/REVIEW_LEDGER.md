# Review Ledger

## Review Scope

This ledger reviews the current Hyperresearch Codex runtime, service topology, proof evidence, and remaining parity claims after the V8 control-plane and packet-provenance implementation loops.

The independent-repository authority correction supersedes earlier findings
whose dispositions treated a personal checkout as Template input or used a
checkout projection as acceptance. Those rows remain audit history only. The
current boundary is an immutable curated-content artifact plus exact versioned
Template interface; no source sync, copied implementation, or coordinated Git
workflow is accepted.
