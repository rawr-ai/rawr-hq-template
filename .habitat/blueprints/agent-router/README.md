# Agent Router

`agent-router` is the lightweight `AGENTS.md` document kind. Every package
boundary owns one instance that locates an operator inside that package,
states only its durable boundaries and flow, links to deeper authority, and
names local proof. It is not architecture authority, project memory, a
target-file inventory, or a replacement for a complete conceptual document.

Habitat structure authority selects package roots from the architecture's
package-kind geometries rather than from one-off names or every internal Nx
project. Grit owns only the positive Markdown anchors and the
repository-relative `AGENTS.md` edge shared by those instances. Authored edges
use standard Markdown so Obsidian and Foam can derive backlinks without
generated backlink lists or ambiguous short wikilinks. Human review owns
whether prose is genuinely local and links are canonical for the concept.
