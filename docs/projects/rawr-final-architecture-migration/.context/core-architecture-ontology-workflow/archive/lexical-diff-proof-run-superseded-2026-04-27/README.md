# Archived Lexical Diff Proof Run

These documents are preserved for historical context but are no longer active guidance.

They were archived after the semantic-diff reframe because they described the Phase 4 testing-plan comparison as a semantic verification run. The current implementation is better understood as lexical triage: it can match ontology labels and aliases in document text, but it does not yet parse comparison documents into evidence claims with polarity, modality, and assertion semantics.
